  Цель проекта:
Создание "домашнего курьера" для помощи маломобильным гражданам и автоматизации бытовых задач. Система позволяет голосом управлять переносом объектов между заранее заданными точками в помещении.
  Основная функциональность:
1) Персонализированное голосовое управление
• Распознавание конкретного пользователя по голосовым признакам
• Фильтрация фоновых звуков (телевизор, разговоры)
• Настройка индивидуальных команд под каждого пользователя
2) Гибкая маршрутизация
• Предварительная запись маршрутов с помощью "обучения"
• Привязка маршрутов к голосовым командам ((Пример: команда "Достань со шкафа" → путь к верхней полке))
  Полный рабочий цикл
1.Вылет с зарядочной станции по голосовой команде
2.Следование по записанному маршруту к точке забора
3.Захват лотка с объектом
4.Доставка к точке назначения
5.Возвращение лотка по команде
6.Коптер летит на зарядочную станцию
